# Changelog

All notable changes to this collection are documented in this file.

## [1.0.0] - 2025-xx-xx

- Initial public release of `lit.supplementary`
- Added `keycloak_config` role for configuring existing Keycloak instances:
  - Realms and clients
  - Realm/client roles and bindings
  - Groups, default groups and users
  - Identity providers (OIDC/SAML) and LDAP/Kerberos federation
  - Realm-wide policies (themes, SMTP, flows, sessions, tokens)
